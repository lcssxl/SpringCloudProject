# SpringCloudProject
基于springcloud搭建的微服务程序


## 使用技术
- 版本：springcloud Camden.SR6
- 注册中心：eureka


## 项目结构
- 注册中心 discovery
- 服务1 com.open.free.service
- 服务2 service2
- 网关 gateway

## 启动项目
1. 启动discovery
2. 启动service
3. 启动service2
4. 启动gateway

## 实现功能
通过注册中心的配置，可以在gateway模块，访问service和service2模块上的接口。
